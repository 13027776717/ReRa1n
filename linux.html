ReRa1n for Linux

This is an install and setup guide for ReRa1n, on Debian based Linux distributions

Downloading:

            
 git clone https://github.com/aidangamzer/rera1n.git
 cd ReRa1n
 chmod +x ./rera1n.sh ./install.sh
 sudo apt-get update

Installing dependencies:

            
 ./install.sh

Running: - Answer all prompts (CaSeSeNsItIvE)

            
 ./rera1n.sh
